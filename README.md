# Linked Data Infortmation Retrieval Project
This repository represents the undertaken work during my research internship at Triply (https://triply.cc/), for my bachelor thesis project at VU Amsterdam.


*NOTE:* This repository does not contain the larger sized datasets, however, methods to create and find these datasets are documented in the notebooks (.ipynb) and paper.

## Evaluation_dataset_and_result_notebooks
This folder contains all the evaluation notebooks and data of the performance results of the information retrieval systems

## Evaluation_scripts
This folder contains all the relevant scripts to conduct the evaluation processes reported in the paper.

## Demonstration_notebooks
This folder illustrates the basic principles of the information retrieval methods used and explains the employed methods in further detail.
